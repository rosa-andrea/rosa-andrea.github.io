P3


P3 is a new profiler suite for parallel applications running on the Java Virtual Machine (JVM), targeting metrics related to parallelism, concurrency, and synchronization.


Demo version

A demo version of P3 can be downloaded here.

The archive contains a virtual machine where P3 has been already set up and is ready to use. Additionally, the virtual machine includes sample workloads that are ready to be profiled with P3.
