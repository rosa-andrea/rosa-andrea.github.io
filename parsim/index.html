ParSim


ParSim allows the reproduction of custom workloads in parallel systems.

With ParSim, users can model workloads of parallel applications through a formal representation. This formalism has been developed to allow the modeling of parallel operations (like forks and joins of child processes) and operating patterns (like CPU- or I/O-intensive instructions).

ParSim can reproduce custom workloads in multicore and multiprocessor systems, generating graphical and textual information for further analysis, with special focus on the parallel behavior of the workload.


Instruction

ParSim only works on the Linux operating system. Every release of ParSim includes a README file that contains instructions about how to compile and use the tool. The package also includes tutorial examples.
